## Exa 指引

### Core Capability
Exa Web 搜索 — 搜索技术文章、最新解决方案、社区讨论。适用于获取超出 LLM 知识截止日期的信息，如最新框架版本变更、新发布的工具、社区最佳实践。需要 API Key 配置。

### Anti-patterns
- 代码搜索 → 用 ace-tool / ContextWeaver
- 库 API 文档 → 用 Context7
- 仓库架构理解 → 用 DeepWiki
- 不适合搜索私有/内部文档

### Query Examples
- `"Latest best practices for Node.js error handling 2026"`
- `"New features in TypeScript 5.9"`
